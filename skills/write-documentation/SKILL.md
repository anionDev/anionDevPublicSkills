Ensure that the documentation of this repository is complete and up-to-date.
This includes the following things:
- A readme-file which gives an overview about the repository, how to use it, how to contribute and so on.
- Doc-comments in the source-code for non-trivial function regarding behavior and design-decisions.
- Documentation in the reference-folder.
If the repository defines any rules/scripts for documentation, etc. then follow them.
